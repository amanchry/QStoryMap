# QStoryMap

This story map was generated by the **QStoryMap** QGIS plugin.

Source code: [github.com/amanchry/QStoryMap](https://github.com/amanchry/QStoryMap)

## View locally

Open this folder with a local web server (opening `index.html` directly may not work in some browsers):

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.
